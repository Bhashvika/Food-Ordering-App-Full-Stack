Project Overview
The Food Ordering App is a full-stack web application that allows users to browse and order food items. The application has three main components: the frontend (user interface), the backend (server and database), and the admin panel (for managing orders and food items). The app also integrates with Stripe for handling payments.

Features
User Authentication: Sign up, login, and logout functionalities.
Browse Food Items: Users can view a list of available food items.
Cart Management: Add, update, or remove items from the cart.
Order Placement: Place orders and make payments using Stripe.
Order Tracking: Track the status of your orders.
Admin Panel: Manage food items, view and update order statuses.

Technologies Used
Frontend: React, React Router, Axios
Backend: Node.js, Express.js, MongoDB
Admin Panel: React
Payment Gateway: Stripe
Authentication: JWT (JSON Web Tokens)
Styling: CSS

Prerequisites
Node.js (v14 or above)
MongoDB
Stripe Account
